# X Spaces Hotkeys Bookmarklet

A powerful bookmarklet for X.com (Twitter) Spaces that lets you assign custom keyboard shortcuts to common actions like muting, leaving quietly, and sending emoji reactions — all without lifting your fingers from the keyboard.

Perfect for frequent Space participants, hosts, or anyone who wants faster control during live audio conversations.

## Features

- **Custom hotkeys** for:
  - Toggle Mute/Unmute
  - Leave Quietly
  - 11 emoji reactions (😂 😲 😢 💜 💯 👏 ✊ 👍 👎 👋 ✋)
- Clean, modern popup with **grid layout** for quick key assignment
- Visual emoji icons with dedicated input fields
- Duplicate key detection
- Persistent storage using `localStorage` — your shortcuts are saved forever
- Works instantly in any live X Space

## Installation

1. Copy the bookmarklet code from the source file: [`bookmarklet.js`](bookmarklet.js)
2. Create a new bookmark in your browser:
   - Right-click the bookmarks bar → **Add page** / **New bookmark**
   - Name it something like **"Spaces Hotkeys"**
   - Paste the code into the URL/address field
3. Save it.

## How to Use

1. Join or host a Space on [x.com](https://x.com)
2. Click your **"Spaces Hotkeys"** bookmark
3. Assign keys by pressing any letter/number (or Space for spacebar)
4. Click **Save & Close**
5. Use your shortcuts instantly!

> Tip: Avoid using keys that conflict with browser shortcuts (like `M` for mute is safe, but `L` might conflict in some contexts).
